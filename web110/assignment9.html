HTML
HTML stands for Hyper Text Markup Language. I like to think of HTML as a language used to accessorize websites. It helps with their appearance and organize the content. There are multiple versions of HTML; HTML 1.0-4.0, XHTML, & HTML5. Tags are orders that tell the web browser how to showcase the text, audio, graphics, or videos. They are created with angle brackets, they end by putting a forward slash in the closing tag. Forms are used to enter values. These are than sent to the server to be processed. CSS stands for Cascading Style Sheets, this helps estblish the presesntation of HTML by creating a separate file with the .css extension. Javascript is a lightweight programing language "with object- oriented capabilities that allows you to build interactivity into otherwise static HTML pages". It is a more advanced website building language. PHP is Hypertext Preprocessor is used for developing web based software application, it is a scripting language that is embedded in HTML. Web developers use it to manage content, databases, session tracking and also building entire e-commerce sites. I think it's so cool how there are multiple coding languages that you can use to design depending on level of expertise.
